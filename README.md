# 人事管理系统简介 python709

## 项目概述

人事管理系统是基于Python语言开发，使用MySQL数据库进行数据存储，前端可选框架为Tkinter和Flask，后端采用MySQL 8.0+进行数据管理。

## 技术栈

- 后端：Python
- 数据库：MySQL 8.0+
- 前端框架（可选）：
 - Tkinter
 - Flask

## 功能模块

### 系统核心功能

- 数据库初始化：自动识别并创建数据库，建立数据模型。
- 用户信息管理：实现登录、添加、修改、删除、查询用户信息。
- 员工信息管理：实现添加、修改、删除、查询员工信息。
- 考勤信息管理：实现添加、修改、删除、查询考勤记录。
- 请假信息管理：实现添加、修改、删除、查询请假信息。

### Flask版独有功能

- 简易防火墙：集成Flask-Limiter限制请求频率，Flask-Talisman增加安全防护。

## 系统角色

系统包含以下角色：

- 系统管理员：负责数据库初始化及用户管理。
- 员工：进行个人信息管理，考勤记录查询。
- 人事经理：负责员工信息管理及请假信息管理。

## 运行环境

- Python环境
- MySQL数据库环境

## 部署步骤

1. 安装Python环境。
2. 安装MySQL数据库，并进行配置。
3. 克隆或下载项目代码，并安装依赖项。
4. 配置数据库连接。
5. 运行系统。
